# Aetherfield

A responsive marketing website for **Aetherfield**, a fictional sustainability/ESG analytics platform. Built from a UI design sourced from the Figma Community as a frontend practice project.

## Overview

Single-page site covering hero, features, values, case study, journal/articles, testimonials, CTA, and footer sections — built with plain HTML and CSS.

### Live Preview

[Live Preview](https://kreemyy.github.io/aetherfield/)

## Tech Stack & Things I Practiced

- HTML5
- CSS3 (Grid, Flexbox, custom properties, scroll-driven animations)
- Vanilla JavaScript (DOM manipulation)
- [Font Awesome](https://fontawesome.com/) for icons
- Google Fonts: Radio Canada Big, Source Serif 4

## Features

- Mobile nav menu: the `+` icon toggles a slide-in sidebar from the right with its own close button to dismiss it.
- Scroll driven animation
